# 图书管理系统 java1145

## 项目概述

本系统是基于 Spring Boot 和 Vue 的前后端分离的图书管理系统。系统主要使用 Java 语言，结合 SpringBoot、MyBatis 和 MySQL 等技术栈进行开发。

## 技术栈

- 前端：Vue
- 后端：Spring Boot
- 数据库：MySQL
- 持久层框架：MyBatis

## 系统角色

系统分为两个角色：普通用户和管理员。

## 功能模块

- **图书信息管理**：管理图书的基本信息。
- **图书类别管理**：对图书的类别进行增删改查操作。
- **借阅信息管理**：查看和管理图书借阅信息。
- **用户管理**：对系统用户进行管理。
- **修改密码**：允许用户更改自己的密码。
- **用户借书**：用户可进行图书借阅操作。
- **用户还书**：用户归还图书。

## 运行环境

- 后端：Java 1.8 或以上版本，兼容 Spring Boot 的运行环境。
- 前端：Vue.js 的兼容浏览器。
- 数据库：MySQL 5.7 或以上版本。
